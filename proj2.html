Studio Ghibli Quote Book

This project was to create a quote book of anything which inspired me. I have always been a fan of Miyazaki and grew up watching the beautiful worlds and adventures which Studio Ghibli animated.

Process

Using brush script and water colour, I tried to make the quotes look child like while using various different colours for each movie theme. Each quote is provided with a page which shows the script where the passage was taken from.
