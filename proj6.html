Plant Life

This project was to create an app which is connected to a smart object, in this case it was smart flower pots.

The App

The app keeps track of the plants health such as it's temperature, water levels, the amount of sunlight being taken in ect. It will then give the user an alert via mobile device if there is anything the plant is lacking. The smart device can be set to auto or manual for convinience.
